# Angular app boilerplate
A simple boilerplate used for starting angular-based web apps utilizing front-end tools like bootstrap and gulp

## Dependencies
- Node & npm
- Bower
- gulp

## Install
```
git clone git@github.com:BjarkeJ/angular-app-boilerplate.git
bower install
npm install
```

## Build
```
gulp
```
